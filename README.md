## Description

A [Nest JS](https://github.com/nestjs/nest) framework + TypeScript + Prisma project.


### To Install Packages

> npm install

### To Run the app

#### development mode
> npm run start

#### watch mode
> npm run start:dev

#### production mode
> npm run start:prod
